# This is made of...

- Heroku
    - web / api server
    - sinatra
- Amazon SQS
    - command queue
- Google Calendar
    - command scheduling
- Chrome App
    - remote controller
- iRemocon
    - universal remote which can be controlled via TCP/IP
- Raspberry Pi
    - raspbian on Model B
